# SAMGEODEZIST MCHJ

SAMGEODEZIST MCHJ uchun Next.js App Router asosidagi korporativ landing sahifa.

## Ishga tushirish

1. `npm install`
2. `npm run dev`
3. Brauzerda `http://localhost:3000` manzilini oching

## Qo'shimcha

- Xarita ko'rsatish uchun `.env.local` fayliga `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=...` qo'shing.
- Telegram buyurtma yuborish uchun `.env.local` fayliga `TELEGRAM_CHAT_ID` va `TELEGRAM_BOT_TOKEN` qo'shing.
- Build tekshiruvi uchun `npm run build` ishlating.
